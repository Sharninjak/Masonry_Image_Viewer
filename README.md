# Masonry Image Viewer
###  瀑布流本地图片浏览器

Source: [wlm3201/Masonry_Image_Viewer: Masonry/Waterfall Image Viewer](https://github.com/wlm3201/Masonry_Image_Viewer)

- [Demo](https://wlm3201.github.io/Masonry_Image_Viewer)

- 建议开启chrome://flags的Smooth Scrolling和Overlay Scrollbars

- 由于Api限制不支持安卓

## 程序简介

Masonry Image Viewer 是一个基于浏览器 File System Access API 的本地图片浏览器，支持：

- 递归加载文件夹内图片
- 瀑布流/横向流布局切换
- 目录树定位跳转
- 图片放大查看与快速翻图
- 按名称/时间/大小排序
- 黑白漫画、纯色边框过滤

## 使用环境

- 推荐浏览器：Chrome / Edge（支持 File System Access API）
- 建议在桌面端系统使用（Windows/macOS/Linux）
- 安卓端由于 API 限制不支持

## 启动方式

1. 直接在浏览器打开 MasonryViewer.html。
2. 首次打开后，点击页面中央提示“点击或拖入图片文件夹”。
3. 选择一个本地目录后，程序会开始递归扫描并按批次加载图片。

说明：

- 也可以把文件夹直接拖拽到页面中导入。
- 支持粘贴文件句柄（系统允许时）。

### 文件列表（新增）

- 左上角新增“+”按钮，可添加图片文件或文件夹。
- 文件列表概念：仅记录你每次添加的最外层文件/文件夹路径（如 `/A`、`/B.jpg`）。
- 点击“+”后：
	- 选择“确定”添加文件夹
	- 选择“取消”添加图片文件（可多选）
- 新增来源会并入当前待加载列表，不会清空现有内容。

### 历史文件列表（新增）

- 程序会自动记录最近使用过的文件列表。
- 在初始“点击或拖入图片文件夹”界面下方会显示历史列表。
- 点击任意历史项即可重新打开该文件列表（首次可能会再次请求文件访问权限）。

## 基础操作

### 1) 加载与滚动

- 页面默认按“每次”设定值进行渐进加载（避免一次性占用过高内存）
- 滚动到接近底部时自动继续加载
- 点击“加载”可强制连续加载全部
- 点击“暂停”可停止连续加载

### 2) 目录树跳转

- 点击左侧目录按钮打开目录树
- 点击某个目录后：
	- 会高亮该目录
	- 自动跳转到该目录的首张已加载图片
	- 若尚未加载到该目录，会继续加载直至命中后跳转
- 跳转后仍可上下滚轮浏览其它目录图片（不会只保留当前目录）

### 3) 图片查看

- 在主视图点击图片进入放大查看模式
- 查看模式下：
	- 左键：上一张
	- 右键：下一张
	- 中键,s：关闭查看
	- 滚轮：缩放
	- ArrowLeft / A：上一张
	- ArrowRight / D：下一张
	- Esc：关闭查看
    - 放大后移动鼠标：拖动图片

## 右侧参数面板说明

点击右上角“+”可打开参数面板。

### 布局相关

- 列距：图片列间距
- 行距：图片行间距
- 圆角：图片圆角半径
- 边框：图片边框宽度
- 列数 + 纵向：瀑布流列数控制
- 高度 + 横向：横向流最小行高控制

### 数据相关

- 总数：扫描到的图片总数量
- 加载：已完成解码的数量
- 显示：当前通过过滤并渲染的数量
- 跳转：输入显示序号后回车可滚动到对应图片
- 每次：每轮加载数量（影响加载速度与流畅度平衡）

### 排序与过滤

- 排序字段：文件名 / 默认 / 日期 / 大小
- 顺序：升序 / 降序
- 重排：按当前排序和过滤条件重新布局
- 黑白漫画：过滤彩色图，保留近似黑白图
- 纯色边框：过滤无明显内容边框图
- 反转：反转过滤结果
- 宽高比：支持区间表达式
	- 16:9（等于）
	- 16:9-（大于）
	- -16:9（小于）
	- 4:3-16:9（区间）

## 常见问题

### 1) 点击后没有弹出目录选择

- 请确认使用支持 File System Access API 的浏览器
- 请确认页面来源允许该 API（本地文件或受信任上下文）

### 2) 目录很多时首次加载较慢

- 属于正常现象，程序会递归扫描并增量加载
- 可降低“每次”值以提升交互流畅度

### 3) 过滤后图片数量明显减少

- 过滤会在显示阶段生效，建议先关闭过滤按钮确认原始结果
